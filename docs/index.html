Obsidian plugins

Obsidian is an excellent Markdown PKM which works off local files. I transitioned to it from Evernote and haven't looked back.

One of the best things about it is that it's super easy to customise if you know a bit of Javascript. These are the plugins I've written for it:

  • Google Photos - Adds Google Photos integration
  • Image Captions - Adds captions to images
  • Folder Notes - Adds Folder Notes to the default file tree
